# Hospital Management System

## مقدمه

این پروژه یک سیستم مدیریت بیمارستانی است که برای مدیریت بیماران، پرسنل و پزشکان طراحی شده است. این سیستم به کاربران این امکان را می‌دهد تا قرار ملاقات‌ها را برنامه‌ریزی کنند، نسخه‌ها را بنویسند، و ارتباطات بین بیماران و پزشکان را مدیریت کنند.

## ویژگی‌ها

- **مدیریت بیماران**: ثبت و مدیریت اطلاعات بیماران، تاریخچه پزشکی، و سوابق درمانی.
- **مدیریت پزشکان**: ثبت و مدیریت اطلاعات پزشکان، تخصص‌ها، و برنامه‌های زمانی.
- **مدیریت پرسنل**: مدیریت اطلاعات پرسنل بیمارستان، از جمله کارکنان اداری و پرستاران.
- **برنامه‌ریزی قرار ملاقات‌ها**: امکان برنامه‌ریزی و مدیریت قرار ملاقات‌های بین بیماران و پزشکان.
- **نوشتن نسخه‌ها**: پزشکان می‌توانند نسخه‌های درمانی برای بیماران بنویسند و تاریخچه نسخه‌ها را مدیریت کنند.

## نصب و راه‌اندازی

### پیش‌نیازها

برای اجرای این پروژه نیاز به موارد زیر دارید:

- Python 3.x
- Django (برای پروژه‌های وب Django)
- پایگاه داده مناسب (مانند SQLite، PostgreSQL، یا MySQL)
